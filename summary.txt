사용자 별 음성 인식 기반 로그인 시스템을 만들기 위해서는 음성 특징 추출,
머신 러닝/딥러닝 모델 학습, 실시간 인증 시스템을 구현해야 합니다.
이를 위해 MFCC, Speaker Recognition 라이브러리, 머신 러닝 알고리즘 등을 사용할 수 있습니다.
처음에는 음성 데이터 수집부터 시작하여 모델 학습을 진행하고,
마지막으로 실시간 음성 인증을 처리하는 방법을 구현해야 합니다.

1. 음성 데이터 수집
2. 음성 특징 추출 - MFCC (Mel Frequency Cepstral Coefficients) 또는 시스템화된 음성 템플릿
3. 음성 인식 시스템 학습 - 수집된 음성 데이터를 바탕으로 사용자의 음성 모델을 학습 , ML
4. 음성 인증 - 사용자가 로그인할 때 실시간 음성을 받아 해당 음성을 인식하고 분석
5. library
- A. SpeechRecognition 라이브러리 (기본 음성 인식), 음성 to 텍스트
- B. pyAudio (오디오 녹음 및 처리) SpeechRecognition
- C. speaker recognition 라이브러리( 음성 생체 인식 시스템을 구현할 수 있습니다.
예를 들어, pyAudioAnalysis 또는 sidekit 등의 라이브러리)

설계 ㅣ 음성 인증 시스템을 구현하려면,
기본적으로 음성 데이터를 특징 추출하고 이를 모델로 학습하여,
로그인할 때마다 실시간으로 음성을 비교하는 과정이 필요
